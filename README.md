
# Oliver Zdraveski Portfolio
Fully customized working front-end website for my Portfolio.


## Run Locally

Clone the project

```bash
  git clone https://github.com/oliverzdraveski/portfolio.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
